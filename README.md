### IIIF Collectionのディレクトリ構造

https://github.com/nakamura196/iiif/tree/master/docs/data/collection

```
docs/data/collection
|   collection.json（本ファイルがすべてのコレクションを包含したIIIF Collectionです）
|   
└───collections
    │   collection_A.json
    │   collection_B.json
    │   collection_C.json
    │   ...

```

```
|       │   ...
│
└───collection_B
|   │   createManifestList.py
|   └───data
|       │   manifest_list.csv
|       │   ...
```
